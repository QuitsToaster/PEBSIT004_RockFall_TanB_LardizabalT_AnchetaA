# **How to Run the Game**

## **Option 1: Run from Unity Editor**

1. **Install Unity Hub** (if not installed)
   üîó [https://unity.com/download](https://unity.com/download)

2. **Open Unity Hub ‚Üí Click ‚ÄúAdd Project‚Äù ‚Üí Select the Rockfall folder**
   *(Example: `Documents/Rockfall`)*

3. Once loaded, **select the scene to play:**

   * Navigate to:

     ```
     Assets/Scenes/
     ```
   * Open the main scene (e.g. `MainScene.unity` or `Rockfall.unity`)

4. Press **Play** on the top toolbar to start the game inside Unity.


## **Option 2: Build and Run (Standalone Version)**

1. In Unity, go to the top menu:

   ```
   File ‚Üí Build Settings
   ```

2. **Select your target platform:**

   * **Windows** ‚Üí `PC, Mac & Linux Standalone`
   * **Mac** ‚Üí `MacOS`
   * **Linux** ‚Üí `Linux`

3. Click **Add Open Scenes**, then **Build and Run**.

4. **Choose a location** to save your build.
   Unity will compile the game and automatically launch it.

| **Shoot**          | `Spacebar` / Tap fire button                     |
