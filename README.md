# PEBSIT004_RockFall_TanB_LardizabalT_AnchetaA

## Project Title
**Rockfall**

## Short Description
A 3D Unity space combat simulator where players defend a central space station from waves of incoming asteroids. Players pilot a spaceship, shoot lasers, and survive multiple short, intense challenges while mastering movement and targeting in a 3D environment.

## Engine / Language / Version
- **Engine:** Unity 3.14.0 
- **Language:** C#  

## Controls
- **Desktop**  
  - W/A/S/D -> Move spaceship
  - Spacebar/Left Mouse Click -> Shoot laser
  - ESC -> Pause/Resume

## How to Run
1. Clone or download this repository:  
   ```bash
   git clone https://github.com/QuitsToaster/PEBSIT004_RockFall_TanB_LardizabalT_AnchetaA.git

2. Open the project in Unity 3.14.0.

3. Load the main scene:

   Assets/Scenes/MainScene.unity

4. Press the Play button in the Unity Editor or build the project:


## Known Issues / Limitations
**Collision response with asteroids may occasionally clip due to Unity's physics timestamp.**
**No save/load system yet (progress resets each play session.**
**Mobile touch input may require calibration on some devices.**
**Audio balancing and explosion effects are still in progress**
